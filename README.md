# 💰 Análise de Dados Financeiros

Projeto desenvolvido para análise de transações financeiras, com foco em extração de insights e manipulação de dados utilizando Python.

## 🚀 Tecnologias

* Python
* Pandas
* Matplotlib
* CSV (base de dados)

## 📊 Funcionalidades

* Cálculo de entradas e saídas
* Análise de saldo final
* Agrupamento de gastos por categoria
* Visualização de dados (gráficos)

## 🎯 Objetivo

Demonstrar habilidades em análise de dados, manipulação de informações financeiras e geração de insights relevantes.

## 💡 Aprendizados

* Análise de dados com Python
* Uso de bibliotecas como Pandas
* Organização e estruturação de projetos
* Visualização de dados

## ▶️ Como executar

1. Instale as dependências:
   pip install pandas matplotlib

2. Execute o script:
   python src/analise.py
