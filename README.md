# 💰 Análise de Dados Financeiros

Projeto de análise de dados financeiros desenvolvido para extrair insights a partir de transações, permitindo identificar padrões de gastos, saldo e comportamento financeiro utilizando Python.

## 🚀 Tecnologias

* Python
* Pandas
* Matplotlib
* CSV (base de dados)

## 📊 Funcionalidades

* Cálculo de entradas e saídas
* Análise de saldo final
* Agrupamento de gastos por categoria
* Visualização de dados (gráficos)

## 🎯 Objetivo

Demonstrar habilidades em análise de dados, manipulação de informações financeiras e geração de insights relevantes.

## 💡 Aprendizados

* Análise de dados com Python
* Manipulação e análise de dados com Pandas
* Organização e estruturação de projetos
* Visualização de dados

## 📈 Exemplo de saída

```bash
Total de entradas: 3500
Total de saídas: 950
Saldo final: 2550

Gastos por categoria:
alimentacao: 150
transporte: 200
lazer: 100
```
